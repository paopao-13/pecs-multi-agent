# 案例模板：计划逻辑遗漏

> 案例编号: 02
> 错误类型: 计划逻辑遗漏 (plan_logic_omission)
> 涉及角色: Planner / Critic / Synthesizer
> 修复机制: Synthesizer 反思 → Planner 重规划 → 补充遗漏步骤

---

## 1. 原始任务

**任务描述**: 计算2024和2020奥运会中国金牌数差值

**期望输出**: 差值为2（2024年40枚 - 2020年38枚 = 2）

---

## 2. 初始错误计划

**Planner 生成的计划**:

```json
[
  {
    "id": 1,
    "description": "搜索2024年巴黎奥运会中国代表团金牌数",
    "tool": "search",
    "tool_input": "2024年巴黎奥运会 中国代表团 金牌数",
    "risk": "medium"
  },
  {
    "id": 2,
    "description": "计算金牌数差值",
    "tool": "python",
    "tool_input": "40 - 38",
    "risk": "low"
  }
]
```

**错误点**: 计划只包含搜索2024年金牌数的步骤，完全遗漏了搜索2020年东京奥运会金牌数的步骤。步骤2中直接硬编码了 "38" 这个未经搜索验证的数字。

---

## 3. 错误执行结果

**Executor 执行步骤 1**:

```
[Executor] 执行步骤 1: 搜索2024年巴黎奥运会中国代表团金牌数 (工具: search)
[Executor] 步骤 1 完成: [模拟搜索] 2024年巴黎奥运会，中国体育代表团获得40枚金牌、27枚银牌、24枚铜牌。
```

**Executor 执行步骤 2**:

```
[Executor] 执行步骤 2: 计算金牌数差值 (工具: python)
[Executor] 步骤 2 完成: 输出:
2
```

**问题分析**: 虽然步骤2的计算结果恰好为2，但这是基于硬编码的 "38" 得出的，而非通过搜索获取的真实数据。计划逻辑存在严重遗漏：缺少搜索2020年东京奥运会金牌数的步骤。

---

## 4. Critic 反思点评

**Critic 对步骤1的评分**:

| 维度 | 分数 | 说明 |
|------|------|------|
| accuracy (准确性) | 4 | 搜索结果准确，包含2024年中国金牌数40枚 |
| consistency (一致性) | 4 | 结果与步骤描述一致 |
| completeness (完整性) | 2 | 仅获取了2024年数据，缺少2020年东京奥运会数据 |
| **overall (综合)** | **3.3** | **步骤本身合格但整体计划不完整** |

**Critic 对步骤2的评分**:

| 维度 | 分数 | 说明 |
|------|------|------|
| accuracy (准确性) | 2 | 计算中使用的"38"未经搜索验证，来源不明 |
| consistency (一致性) | 3 | 计算逻辑正确，但输入数据不可靠 |
| completeness (完整性) | 1 | 完全缺少2020年东京奥运会金牌数的搜索步骤 |
| **overall (综合)** | **2.0** | **不达标，需要补充搜索步骤** |

**Critic 反馈**:

```
[Critic] 规则验证步骤 2: 评分 2.0 (计算使用的2020年金牌数未经搜索验证，计划缺少搜索2020年东京奥运会中国代表团金牌数的步骤。)
[Critic] 步骤 2 不合格，反馈: 计划逻辑遗漏：缺少搜索2020年东京奥运会中国代表团金牌数的步骤。当前步骤2中"38"为硬编码值，未经工具验证。建议增加搜索2020年东京奥运会金牌数的步骤，确保数据来源可靠。
```

**拦截状态**: 已拦截，触发 Synthesizer 反思

---

## 5. 修正后计划

**Synthesizer 反思 → Planner 重规划**:

```
[Synthesizer] 触发反思循环 (第 1 轮): 有 1 个步骤不完整，计划缺少搜索2020年东京奥运会中国代表团金牌数的步骤，需要补充搜索步骤后重新计算。
```

**Planner 重规划后的计划**:

```json
[
  {
    "id": 1,
    "description": "搜索2024年巴黎奥运会中国代表团金牌数",
    "tool": "search",
    "tool_input": "2024年巴黎奥运会 中国代表团 金牌数",
    "risk": "medium"
  },
  {
    "id": 2,
    "description": "搜索2020年东京奥运会中国代表团金牌数",
    "tool": "search",
    "tool_input": "2020年东京奥运会 中国代表团 金牌数",
    "risk": "medium"
  },
  {
    "id": 3,
    "description": "计算两次金牌数差值",
    "tool": "python",
    "tool_input": "gold_2024 - gold_2020",
    "risk": "low"
  }
]
```

**修正说明**: 新增了步骤2（搜索2020年东京奥运会中国代表团金牌数），并将原步骤2的计算移至步骤3，使用变量引用而非硬编码。
