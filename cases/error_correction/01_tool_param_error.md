# 案例模板：工具参数错误

> 案例编号: 01
> 错误类型: 工具参数错误 (tool_param_error)
> 涉及角色: Executor / Critic
> 修复机制: Critic 拦截 → executor_retry → 正确参数重新调用

---

## 1. 原始任务

**任务描述**: 搜索2024年巴黎奥运会中国金牌数

**期望输出**: 40枚金牌

---

## 2. 初始错误计划

**Planner 生成的计划**:

```json
[
  {
    "id": 1,
    "description": "搜索2024年巴黎奥运会中国金牌数",
    "tool": "search",
    "tool_input": "巴黎奥运会 金牌",
    "risk": "medium"
  }
]
```

**错误点**: Executor 使用了模糊关键词 "巴黎奥运会 金牌"，缺少年份限定 "2024" 和国家限定 "中国"，导致搜索结果不精准。

---

## 3. 错误执行结果

**Executor 执行步骤 1**:

```
[Executor] 执行步骤 1: 搜索2024年巴黎奥运会中国金牌数 (工具: search)
[Executor] 步骤 1 完成: [模拟搜索] 2024年巴黎奥运会（第33届夏季奥林匹克运动会）于2024年7月26日至8月11日在法国巴黎举行。共有206个国家和地区参赛，共产生329枚金牌。美国以40枚金牌位居榜首...
```

**问题分析**: 搜索返回的是巴黎奥运会总体金牌信息，而非中国代表团的金牌数。参数中缺少 "中国" 关键词，导致结果偏向赛事总览而非中国代表团。

---

## 4. Critic 反思点评

**Critic 评分**:

| 维度 | 分数 | 说明 |
|------|------|------|
| accuracy (准确性) | 2 | 搜索结果未直接回答"中国金牌数"，返回的是赛事总览信息 |
| consistency (一致性) | 3 | 结果与任务主题相关，但内容不匹配具体需求 |
| completeness (完整性) | 1 | 完全缺少中国代表团的金牌数据 |
| **overall (综合)** | **2.0** | **不达标，需要重试** |

**Critic 反馈**:

```
[Critic] 规则验证步骤 1: 评分 2.0 (搜索参数缺少"中国"关键词，结果未包含中国代表团金牌数据。)
[Critic] 步骤 1 不合格，反馈: 搜索参数缺少"中国"关键词，结果未包含中国代表团金牌数据。建议使用更精确的关键词重新搜索，包含"2024"、"巴黎奥运会"、"中国"、"金牌数"。
```

**拦截状态**: 已拦截，触发 executor_retry

---

## 5. 修正后计划

**Critic 反馈触发的修正**:

```json
[
  {
    "id": 1,
    "description": "搜索2024年巴黎奥运会中国金牌数",
    "tool": "search",
    "tool_input": "2024年巴黎奥运会 中国代表团 金牌数",
    "risk": "medium",
    "retry_count": 1
  }
]
```

**修正说明**: 在搜索关键词中增加了 "2024"、"中国代表团" 和 "金牌数"，使搜索更精准。

---

## 6. 正确输出

**Executor 重试执行步骤 1**:

```
[Executor] 重试步骤 1 (第 1 次重试)
[Executor] 执行步骤 1: 搜索2024年巴黎奥运会中国金牌数 (工具: search)
[Executor] 步骤 1 完成: [模拟搜索] 2024年巴黎奥运会，中国体育代表团获得40枚金牌、27枚银牌、24枚铜牌，金牌数排名第二。
```

**Critic 二次评审**:

```
[Critic] 规则验证步骤 1: 评分 4.0 (搜索结果有效，包含中国代表团金牌数40枚。)
[Critic] 步骤 1 合格，进入下一环节
```

**Synthesizer 最终输出**:

```
[Synthesizer] 开始综合 1 个步骤的结果
[Synthesizer] 抽取式综合完成（确定性/预算感知路径）
最终答案: 40
```

**正确答案**: 40枚金牌
